# vim-sl

![SL](https://raw.githubusercontent.com/mattn/vim-sl/master/screenshot.gif)

## Usage

```vim
call sl#animate()
```

## Installation

To install using [Vim-Plug](https://github.com/junegunn/vim-plug):

```
" add this line to your .vimrc file
Plug 'mattn/vim-sl'
```
